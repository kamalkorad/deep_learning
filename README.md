# Deep Learning Projects

A collection of deep learning projects applied to real-world healthcare and finance datasets. Each project is self-contained and covers data loading, exploratory analysis, preprocessing, model building, training, and evaluation.

---

## Projects

| # | Project | Type | Dataset | Model |
|---|---------|------|---------|-------|
| 1 | Heart Disease Prediction | Binary classification — Tabular | Cleveland Heart Disease Dataset (303 records) | Feedforward Neural Network |
| 2 | Pneumonia Detection from Chest X-Rays | Binary classification — Image | PneumoniaMNIST (5,856 grayscale images) | CNN |
| 3 | Skin Lesion Classification | Multi-class classification — Image | DermaMNIST (10,015 RGB images, 7 classes) | CNN |
| 4 | Credit Card Fraud Detection | Binary classification — Tabular | Credit Card Fraud Dataset (284,807 transactions) | Feedforward Neural Network |

---

## Tools and Libraries
`Python` · `TensorFlow / Keras` · `Scikit-learn` · `Pandas` · `NumPy` · `Matplotlib` · `Seaborn` · `MedMNIST`

All notebooks are designed to run end-to-end on Google Colab with no external API credentials or manual dataset downloads required.
